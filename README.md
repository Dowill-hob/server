## 🖥️ 프로젝트 소개
간단한 1대1 실시간 채팅 서버를 구현하는 프로젝트입니다.

#### 프로젝트 구동법<br>
파일 여는법 zip해제후 파일을 연다 --> x64폴더 --> DEBUG --> exe 파일 실행

* server 파일의 exe파일을 가장 먼저 구동 시킨다.
* 그 후 클라이언트 exe를 구동 시키면 작동 준비는 끝이다.
